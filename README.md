 🐲 Night Fury – Zelda Variant

This section covers the Legend of Zelda-themed **Night Fury** dragon skins for the Isle of Berk mod.

---

## 🐺 Wolf Link
Summon the Wolf Link version of the Night Fury with the following command:

```
/summon isleofberk:night_fury ~ ~ ~ {VariantName:wolf_link}
```
